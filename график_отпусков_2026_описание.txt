ГРАФИК ОТПУСКОВ НА 2026 ГОД
============================

СТРУКТУРА ФАЙЛА:
---------------

1. ЛИСТ "График отпусков 2026"
   - Заголовок с календарем (числа и дни недели по месяцам)
   - Столбцы A-D: №, ФИО сотрудника, Должность, Отдел
   - Столбцы E-...: Дни 2026 года (с января по декабрь)
   - Цветовая схема:
     * Белый - рабочие дни
     * Серый - выходные
     * Красный - праздничные дни
     * Желтый - предпраздничные дни

2. ЛИСТ "Сотрудники"
   - Столбцы:
     * №
     * ФИО сотрудника
     * Должность
     * Отдел
     * Периоды отпусков (объединенная ячейка)
       - Подзаголовки: Начало отпуска, Конец отпуска
   - Каждому сотруднику по 2 строки для ввода периодов отпусков

3. ЛИСТ "Легенда"
   - Обозначения цветов и типов дней
   - Статистика 2026 года

4. ЛИСТ "Праздники"
   - Перечень праздничных дней 2026 года

ПРОИЗВОДСТВЕННЫЙ КАЛЕНДАРЬ 2026 ГОДА:
------------------------------------

ПРАЗДНИЧНЫЕ ДНИ (ст. 112 ТК РФ):
- 01.01.2026 - 08.01.2026: Новогодние каникулы (8 дней)
- 07.01.2026: Рождество Христово
- 23.02.2026: День защитника Отечества
- 08.03.2026: Международный женский день
- 01.05.2026: Праздник Весны и Труда
- 09.05.2026: День Победы
- 12.06.2026: День России
- 04.11.2026: День народного единства

ПРЕДПРАЗДНИЧНЫЕ ДНИ (сокращенный рабочий день):
- 20.02.2026: перед 23.02.2026
- 07.03.2026: перед 08.03.2026
- 08.05.2026: перед 09.05.2026
- 11.06.2026: перед 12.06.2026
- 03.11.2026: перед 04.11.2026
- 31.12.2026: перед 01.01.2027

СИСТЕМА ОБОЗНАЧЕНИЙ В КАЛЕНДАРЕ:
---------------------------------

- Рабочие дни: число + сокращение дня недели (Пн, Вт, Ср, Чт, Пт)
- Выходные: число + сокращение дня недели (Сб, Вс)
- Праздничные: число + символ "✶"
- Предпраздничные: число + символ "◐"

ПРИМЕР ЗАПОЛНЕНИЯ ЛИСТА "СОТРУДНИКИ":
------------------------------------

№ | ФИО сотрудника           | Должность   | Отдел         | Периоды отпусков
  |                          |             |               | Начало отпуска | Конец отпуска
1 | Иванов Иван Иванович     | Менеджер    | Отдел продаж  | 01.06.2026     | 14.06.2026
  |                          |             |               | 01.12.2026     | 14.12.2026
2 | Петров Петр Петрович     | Разработчик | IT отдел      | 15.07.2026     | 28.07.2026
  |                          |             |               | 15.01.2027     | 28.01.2027

ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ:
---------------------------

1. На листе "Сотрудники" заполните данные сотрудников:
   - Введите ФИО, должности и отделы
   - Укажите периоды отпусков для каждого сотрудника (по 2 строки на сотрудника)

2. На листе "График отпусков 2026":
   - Введите данные сотрудников в столбцы A-D
   - Отметьте периоды отпусков в соответствующих ячейках календаря
   - Используйте цветовую схему для обозначения типов дней

3. Для автоматического заполнения календаря:
   - Скрипт будет читать данные с листа "Сотрудники"
   - Автоматически отмечать периоды отпусков на календаре
   - Проверять пересечения отпусков сотрудников

ОСОБЕННОСТИ РЕАЛИЗАЦИИ:
----------------------

1. Каждому сотруднику положено 28 календарных дней отпуска в год
2. Один отпуск должен быть не менее 14 календарных дней
3. Рекомендуется планировать отпуска с учетом производственных потребностей
4. Не допускается предоставление отпусков в периоды пиковых нагрузок без согласования

ПРИМЕЧАНИЯ:
----------

1. Файл должен быть создан с использованием библиотеки openpyxl для Python
2. Для работы необходимы установленные библиотеки: openpyxl, pandas
3. Скрипт должен запускаться командой: python vacation_scheduler.py
4. Результатом работы будет файл "график_отпусков_2026.xlsx"
